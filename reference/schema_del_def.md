# Delete a schema definition

Delete a schema definition

## Usage

``` r
schema_del_def(x, name, error_if_missing = TRUE)
```

## Arguments

- x:

  A `SchemaDoc`.

- name:

  Definition name to remove.

- error_if_missing:

  Logical flag indicating whether a missing definition should raise an
  error.

## Value

A modified `SchemaDoc`.
